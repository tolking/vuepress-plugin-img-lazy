# 介绍

一个为了更好的支持图片懒加载的 vuepress 插件

该插件将优先使用 原生 的图片懒加载，如果浏览器不支持将通过 lozad 实现

基于 markdown-it-img-lazy、 markdown-it-imsize、 lozad

# 安装

yarn add vuepress-plugin-img-lazy
# or
npm i vuepress-plugin-img-lazy

# 使用

module.exports = {
  plugins: [
    'img-lazy'
  ]
}

![img](/img.jpg)
<!-- or -->
![img](/img.jpg =500x300) <!-- 最佳 -->

# 在主题中使用 ^1.0.1

  • 全局注册
// enhanceAppFile.js
import ImgLazy from 'vuepress-plugin-img-lazy/ImgLazy'

export default ({ Vue }) => {
  Vue.component(ImgLazy.name, ImgLazy)
}
  • 或者直接引用
import ImgLazy from 'vuepress-plugin-img-lazy/ImgLazy'

export default {
  components: { ImgLazy }
}
  • 使用
<template>
  <img-lazy src="/img.jpg" />
</template>

# 配置

# useLoading

  • Type: Boolben
  • Default: true

是否使用基于原生的懒加载

# selector

  • Type: string
  • Default: lazy

默认的懒加载类名

# 其它说明

  1. 已经默认包含 基础路径，但不包括 markdown 文件里面的 <img/> 标签 ^1.0.2

如果需要同时使用 基础路径 与 <img/> 标签，请参考

<img :data-src="$withBase('/img.png')" loading="lazy" class="lazy">
